### STEPS:

Clone the repository

```bash
https://github.com/YemineniPavan/kidney_tumor_classification
```

```bash
python -m venv kidney
```

```bash
kidney\Scripts\activate
```


### STEP 02- install the requirements
```bash
pip install -r requirements.txt
```
